# Click-It Writer

A simple word processor for young learners.

## GitHub Pages Deployment

This project is configured to potentially deploy to GitHub Pages using GitHub Actions.

**To enable deployment:**

1.  Go to **Settings** > **Pages** in this repository.
2.  Under **Build and deployment**, ensure **Source** is set to **GitHub Actions**.
3.  The deployment workflow (`.github/workflows/deploy.yml`) will automatically build and deploy the site on every push to the `main` branch.
4.  Once deployed, your site will be available at: `https://davisgulf7.github.io/simiple-writer/` (or similar, check the Actions tab for the URL).
